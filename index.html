在线接收 Outlook 邮箱验证码

粘贴一行 `email----password----client_id----refresh_token` 格式的账号信息，点击获取即可查看最近验证码。网页通过 boji1334 后端调用 Outlook OAuth2 / IMAP，不会把 token 放进地址栏。

验证码入口

默认只返回最近 30 秒内的 OpenAI 验证码。程序也可以直接调用后端 `POST /api/v1/code` 获取 code 字段。

接口使用

客户查看 打开本页面，粘贴账号信息后点击获取验证码。
程序调用 POST /api/v1/code，直接读取 JSON 里的 code 字段。
安全习惯 账号 token 不进 URL，减少浏览器历史和访问日志泄露。
